The Python Code Starts with...


              import json


              def generate_number_list(start_number, end_number):
                  number_list = []
                  for number in range(start_number, end_number):
                      number_list.append(number)

                  return number_list


              def lambda_handler(event, context):
                  start_number = int(event['queryStringParameters']['start_number'])
                  end_number = int(event['queryStringParameters']['end_number'])

                  number_list = generate_number_list(start_number, end_number + 1)

                  numbers_response = {'number_list': number_list}

                  request_response = dict(statusCode=200, headers={})
                  request_response['headers']['Content-Type'] = 'application/json'
                  request_response['body'] = json.dumps(numbers_response)

                  return request_response

The Javascript Code

The Javascript Starts with the function getNumbers which retrieves the values from the web form using document.getElementById

To ensure and validate the values are integers, the inputs are parsed with parseInt and following with a conditional to check value type using Number.isInteger

If the input returns TRUE the numbers are generated and then displayed using the functions generateNumbers and displayNumbers respectively otherwise an alert is returned





The generateNumbers functions takes in both values and generates values for an empty list. The numbers are iterated over and pushed to the empty list which is then returned







To display the numbers to our app with displayNumbers the function takes the numbers array returned from the previous function. These numbers are then put through another loop that creates a template literal for generating the table rows.







After each iteration, the method document.getElementById("results").innerHTML and assigning the values for each literal


              function getNumbers() {
                  let startNumber = document.getElementById("startNumber").value;
                  let endNumber = document.getElementById("endNumber").value;

                  startNumber = parseInt(startNumber);
                  endNumber = parseInt(endNumber);

                  if (Number.isInteger(startNumber) && Number.isInteger(endNumber)) {
                      generateNumberArray(startNumber, endNumber);

                  } else {
                      alert("You Must Enter A Valid Integer");
                  }
              }

              function generateNumberArray(startNumber, endNumber) {
                  const url = `https://api.joeterlecki.io/0-to-100?start_number=${startNumber}&end_number=${endNumber}`
                  fetch(url)
                      .then((response) => {
                          return response.json();
                      })
                      .then((data) => {
                          let numbers = data["number_list"];
                          let templateRows = "";
                          for (let number = 0; number < numbers.length; number++) {
                              let displayNumber = numbers[number];
                              if (displayNumber % 2 === 0) {
                                  templateRows += `${displayNumber}`;
                              } else {
                                  templateRows += `${displayNumber}`;
                              }
                          }
                          document.getElementById("results").innerHTML = templateRows;
                      })
              }
